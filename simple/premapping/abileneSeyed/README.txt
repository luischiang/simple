Premapping folder required files:

Chains:
filename: 	chains.txt
In this file we need to write the chains that the networks requires to traverse, example:

N1 N2 IDS,FIREWALL 100
******************************************************************************************
Hosts:
filename: net-mb-info.txt
In this file is described the networks in the topology, and the middle boxes

******************************************************************************************
Middle Box Inventory:
filename: mboxes_inventory.txt
In this files is described each middle box, their function

******************************************************************************************
Net and MiddleBox access ports:
filename: net-mb-ports.txt
Indicates where is connected each network or middle box to the network

******************************************************************************************
Switches inventory:
filename: switches.txt

Indicates the switches in the network, including their ids
